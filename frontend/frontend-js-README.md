# How to install and get a nodejs server running

First make sure that you have installed nodejs and npm 
on your computer and that their versions are higher 
than the following respectively

```
> node --version
v21.5.0
> npm --version
10.2.4
> npx --version
10.2.4
> yarn --version
1.22.21
```
